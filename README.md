# QuantityMeasurementApp

QuantityMeasurementApp is a Java application developed to compare, validate, and convert different units of measurement such as feet, inches, yards, kilograms, grams, liters, and gallons. The purpose of this project is to build a reliable and reusable system for handling quantity measurements accurately using object-oriented programming concepts.

The main objective of the project is to check equality between quantities, perform unit conversions, support arithmetic operations between compatible units, and reject invalid comparisons. For example, the application can verify that 1 foot is equal to 12 inches or that 1000 grams is equal to 1 kilogram.

This project follows a hybrid development methodology using both Design-Develop-Test (DDT) and Test Driven Development (TDD). High-level planning such as class design, architecture, and use case identification is completed first. Then each feature is implemented using TDD by writing test cases first, writing minimum code to pass the tests, and refactoring the code for better quality.

The application is implemented in Java and managed using Git and GitHub. Separate branches are created for each use case, and code changes are merged into the main branch through pull requests. This workflow helps maintain clean version control and organized development.

Some major use cases in the project include checking feet equality, inch equality, feet-to-inch conversion equality, yard-to-feet equality, kilogram-to-gram equality, and gallon-to-liter equality. The project can also be extended further to include addition of quantities and more measurement categories.

Testing is an important part of the project. Test cases verify same-value equality, different-value inequality, null handling, conversion accuracy, invalid comparisons, and edge cases. This ensures the application behaves correctly and remains easy to maintain.

Overall, QuantityMeasurementApp is a practical project that demonstrates Java programming, object-oriented design, unit testing, Git workflow, and real-world problem solving through quantity measurement logic.
